```markdown
# 🎬 CTG Movie Search Telegram Bot

এটি একটি অত্যন্ত প্রফেশনাল, অ্যাসিনক্রোনাস (Asynchronous) এবং মডুলার টেলিগ্রাম মুভি ফিল্টার ও সার্চ বট। এই বটটি ডাটাবেজ হিসেবে **MongoDB Atlas** ব্যবহার করে এবং হোস্টিং প্ল্যাটফর্ম (যেমন: Render, Heroku) এর সাথে সামঞ্জস্যপূর্ণ রাখতে এতে একটি লাইটওয়েট ব্যাকগ্রাউন্ড ওয়েব সার্ভার যুক্ত করা হয়েছে।

## 📁 প্রজেক্ট ফোল্ডার স্ট্রাকচার (Project Structure)

বটটির কোডগুলো হিবিজিবি এড়াতে অত্যন্ত সহজ ও মডুলার উপায়ে সাজানো হয়েছে:

```text
movie_bot/
│
├── config.py             # কনফিগারেশন ও ক্রেডেন্সিয়াল ফাইল
├── database.py           # মঙ্গোডিবি ডাটাবেজ কানেকশন ও অপারেশন ফাইল
├── bot.py                # বটের মেইন এন্ট্রি পয়েন্ট এবং ওয়েব সার্ভার পোর্ট বাইন্ডার
├── requirements.txt      # প্রয়োজনীয় ডিপেন্ডেন্সি বা লাইব্রেরির তালিকা
│
└── plugins/              # প্লাগইন ফোল্ডার (বটের মূল ফিচারসমূহ)
    ├── start.py          # স্বাগত বার্তা ও ইউজার গাইড মডিউল
    ├── search.py         # মুভি সার্চ ও ফাইল ডেলিভারি মডিউল
    ├── index.py          # রিয়েল-টাইম অটো এবং লাইভ লোডার বাল্ক ইনডেক্সিং মডিউল
    └── admin.py          # এডমিন কন্ট্রোল প্যানেল (Stats, Broadcast, Delete)
```

## ⚙️ কোন ফাইলের কী কাজ এবং বিস্তারিত বিবরণ

### ১. `config.py` (কনফিগারেশন ফাইল)
*   **কাজ:** এই ফাইলে বটের সমস্ত সিক্রেট টোকেন, এপিআই কি, মঙ্গোডিবি লিঙ্ক এবং এডমিন আইডি সংরক্ষিত থাকে। 
*   **কেন গুরুত্বপূর্ণ:** বটের কোডে সরাসরি টোকেন না লিখে এখানে রাখায় কোডের নিরাপত্তা বজায় থাকে।

### ২. `database.py` (ডাটাবেজ হ্যান্ডলার)
*   **কাজ:** এটি মঙ্গোডিবির সাথে সংযোগ তৈরি করে। এতে ইউজার সেভ করা, মুভি ইনডেক্স করা, সার্চ কুয়েরি চালানো এবং এডমিন ডিলিট অপশনের জন্য ব্যাকগ্রাউন্ড ফাংশনগুলো থাকে।
*   **কেন গুরুত্বপূর্ণ:** এখানে `motor` লাইব্রেরি ব্যবহার করা হয়েছে, যা সম্পূর্ণ নন-ব্লকিং (Async)। ফলে হাজার হাজার ইউজার সার্চ করলেও ডাটাবেজ কখনো স্লো হবে না।

### ৩. `bot.py` (প্রধান রানার ফাইল)
*   **কাজ:** এটি বটকে স্টার্ট করে এবং পাইথনের নতুন ভার্সনগুলোর (যেমন Python 3.11+) ইভেন্ট লুপ এরর দূর করে। এছাড়া Render-এর জন্য ব্যাকগ্রাউন্ডে একটি মিনি পোর্ট-বাইন্ডিং ওয়েব সার্ভার চালায়।
*   **কেন গুরুত্বপূর্ণ:** এটি ছাড়া Render প্ল্যাটফর্মের ফ্রি টিয়ারে বট হোস্ট করলে পোর্ট টাইমআউট এররের কারণে বট বন্ধ হয়ে যেত।

### ৪. `requirements.txt` (লাইব্রেরির তালিকা)
*   **কাজ:** প্রজেক্ট রান করার জন্য প্রয়োজনীয় লাইব্রেরিগুলোর (`pyrogram`, `tgcrypto`, `motor`, `dnspython`) নাম ধারণ করে।

### ৫. `plugins/start.py` (স্টার্ট কমান্ড মডিউল)
*   **কাজ:** ইউজার বটের ইনবক্সে `/start` দিলে তাকে একটি সুন্দর প্রফেশনাল ওয়েলকাম নোটিশ দেখায়। 
*   **কেন গুরুত্বপূর্ণ:** এটি অত্যন্ত নিরাপদ। ডাটাবেজ স্লো থাকলেও ইউজারকে প্রথমে মেসেজ পাঠিয়ে দেবে এবং ব্যাকগ্রাউন্ডে ইউজার আইডি ডাটাবেজে সেভ করবে, ফলে বট কখনো হ্যাং হবে না।

### ৬. `plugins/search.py` (সার্চ ইঞ্জিন ও ফাইল ডেলিভারি)
*   **কাজ:** ইউজার কোনো মুভির নাম লিখে সার্চ করলে মঙ্গোডিবি থেকে রেগুলার এক্সপ্রেশন (Regex) দিয়ে দ্রুত ফলাফল এনে বাটন আকারে দেখায়। বাটনে ক্লিক করলে মুভিটির সাথে সুন্দর টাইটেল এবং সাইজের ক্যাপশন সহ ফাইল ডেলিভারি করে।
*   **কেন গুরুত্বপূর্ণ:** এতে ভিডিও এবং ডকুমেন্ট উভয় ফাইল সরাসরি পাঠানো যায়। কোনো প্রকার `Expected DOCUMENT, got VIDEO` জাতীয় এরর আসে না।

### ७. `plugins/index.py` (অটো ও বাল্ক ইনডেক্সিং মডিউল)
*   **কাজ:** 
    *   **অটো ইনডেক্স:** মেইন চ্যানেলে কোনো ফাইল আপলোড দিলে বট সাথে সাথে স্বয়ংক্রিয়ভাবে সেভ করে নেয়।
    *   **বাল্ক ইনডেক্স:** এডমিন অন্য যেকোনো চ্যানেল থেকে শেষ মেসেজটি ফরোয়ার্ড করলে বট পেছনের সমস্ত মেসেজ স্ক্যান করে ডাটাবেজে সেভ করে। স্ক্যানিংয়ের সময় এটি স্ক্রিনে **লাইভ কাউন্টার লোডার** প্রদর্শন করে।

### ৮. `plugins/admin.py` (এডমিন কন্ট্রোল প্যানেল)
*   **কাজ:** শুধুমাত্র এডমিন আইডি ব্যবহার করে এই স্পেশাল কমান্ডগুলো চালানো যাবে:
    *   `/stats` : বটের লাইভ ইউজার এবং মুভির সংখ্যা দেখা।
    *   `/delete [মুভির নাম]` : ডাটাবেজ থেকে নির্দিষ্ট মুভি ডিলিট করা।
    *   `/clean_database` : সম্পূর্ণ ডাটাবেজ রিসেট বা খালি করা।
    *   `/broadcast` (যেকোনো মেসেজে রিপ্লাই দিয়ে) : এক ক্লিকে বটের সব ইউজারের কাছে নোটিশ বা ফাইল ব্রডকাস্ট করা।

## 🚀 কিভাবে রান করবেন? (Local Setup)

১. প্রথমে প্রয়োজনীয় লাইব্রেরিগুলো ইনস্টল করুন:
```bash
pip install -r requirements.txt
```

২. `config.py` ফাইলে আপনার টেলিগ্রাম ক্রেডেন্সিয়াল ও মঙ্গোডিবি কানেকশন স্ট্রিং বসান।

৩. রান করুন:
```bash
python bot.py
```

## ☁️ Render-এ হোস্ট করার নিয়ম (Deployment)

1. আপনার গিটহাব রিপোজিটরিতে কোডগুলো পুশ করুন।
2. Render-এ একটি নতুন **Web Service** তৈরি করুন এবং আপনার গিটহাবের সাথে কানেক্ট করুন।
3. **Build Command:** `pip install -r requirements.txt`
4. **Start Command:** `python bot.py`
5. Render Environment Variables-এ গিয়ে `PYTHON_VERSION` সেট করুন `3.10.11` অথবা `3.11.8`।
6. ডিপ্লয় করুন এবং ২৫-২৬ মিনিটের টেলিগ্রাম ফ্লাড ওয়েট (যদি থাকে) শেষ হওয়া পর্যন্ত অপেক্ষা করুন।
